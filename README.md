Бот с цитатами:

Telegram-бот для случайных цитат.

Как запустить:

1. Скачайте проект с GitHub
2. Установите Python (python.org)
3. Откройте командную строку в папке проекта
4. Установите библиотеки: pip install -r requirements.txt
5. Получите токен бота у @BotFather
6. Создайте файл .env и добавьте: TELEGRAM\_BOT\_TOKEN=ваш\_токен
7. Запустите бота: python bot.py
8. Напишите боту в Telegram: /start

Команды бота:

/start - начало работы
/quote - случайная цитата
/tags - все теги
/add - добавить цитату
/help - помощь

Файлы проекта:

bot.py - главный файл бота
database.py - база данных
quotes.py - работа с цитатами
